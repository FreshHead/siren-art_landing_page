# Siren-Art Website

Профессиональная аэрография для вашего бизнеса

## Структура проекта

```
siren-art/
├── index.html          # Основная страница
├── style.css           # Стили
├── script.js           # JavaScript функциональность
├── images/             # Изображения
│   ├── truck-hero.png
│   ├── warehouse.png
│   ├── alfa-building.png
│   ├── artist.png
│   └── whatsapp-icon.png
└── README.md
```

## Функциональность

- Адаптивный дизайн для всех устройств
- Плавная прокрутка по секциям
- WhatsApp интеграция для связи
- Анимации и эффекты при прокрутке
- Оптимизированная загрузка изображений

## Настройка изображений

Поместите следующие изображения в папку `images/`:
- `truck-hero.png` - главное изображение грузовика
- `warehouse.png` - склад с техникой
- `alfa-building.png` - здание АЛЬФА
- `artist.png` - художник за работой
- `whatsapp-icon.png` - иконка WhatsApp

## Запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
python -m http.server 8000
```

Затем откройте http://localhost:8000
